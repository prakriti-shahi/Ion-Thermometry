Place all Pluto notebooks in the same directory as IonSim_Thermometry.ipynb, and then run IonSim_Thermometry.ipynb. The interactive plots will automatically launch the corresponding Pluto notebooks in a web browser to perform the full thermometry analysis workflow.

This repository contains simulations of Rabi flopping of a single trapped ion in a Paul trap as temperature and other experimental parameters are varied. The analysis enables exploration of how motional excitation and temperature affect coherent dynamics, supporting thermometry studies in trapped-ion experiments.
